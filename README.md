# To-Do List + Fokus Belajar (Android Kotlin)

## 📱 Deskripsi
Aplikasi untuk mencatat tugas dan membantu fokus belajar menggunakan timer.

## ✨ Fitur
- Tambah tugas
- Lihat daftar tugas
- Detail tugas
- Timer fokus belajar (countdown)
- Validasi input
- Animasi sederhana

## 🧱 Teknologi
- Kotlin
- Android Studio
- RecyclerView
- CardView
